# AC Awake Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a Tauri + React + Redux Toolkit macOS app that prevents system sleep while AC power is connected.

**Architecture:** The React UI owns user-visible state through Redux Toolkit. The Tauri backend owns the sleep-prevention process and exposes `start_awake`, `stop_awake`, and `awake_status` commands.

**Tech Stack:** Tauri v2, Vite, React, TypeScript, Redux Toolkit, React Redux, Vitest, Rust.
