# Sleepless 0.1.1

Maintenance release focused on making Sleepless behave more like a proper macOS menu bar utility.

## Download

- `Sleepless_0.1.1_aarch64.dmg`

This build is for Apple Silicon Macs.

## Changes

- Hide the Dock icon when the main window is closed.
- Keep the menu bar icon alive when the window is closed.
- Treat normal app quit requests such as `Cmd+Q` like hiding the window instead of fully quitting the app.
- Keep the tray menu `종료` item as the explicit full-exit path.
- Bump app version from `0.1.0` to `0.1.1`.
- Update README download links for the new release artifact.

## macOS Security Notice

This build may be unsigned and not notarized. macOS may show an unidentified developer warning on first launch.

If the app is blocked, open it from Finder with right-click > Open, or allow it from System Settings > Privacy & Security.

## Recovery

If the app exits unexpectedly and closed-lid sleep behavior is not restored, run:

```bash
sudo pmset -c disablesleep 0
```
