# Sleepless 0.1.0

First public test release of Sleepless, a small macOS menu bar app that helps keep a Mac awake while connected to AC power.

## Download

- `Sleepless_0.1.0_aarch64.dmg`

This build is for Apple Silicon Macs.

## Highlights

- Keep-awake mode for AC power use.
- Menu bar control with Korean menu labels.
- 8bit ghost app and tray icons.
- Closed-lid duration options: 10 minutes, 30 minutes, 1 hour, or forever.
- Automatic shutdown when AC power is disconnected.
- Battery safety checks while the lid is closed.
- Login item option for background menu bar launch.
- Recovery command documented for restoring macOS sleep behavior.

## macOS Security Notice

This build may be unsigned and not notarized. macOS may show an unidentified developer warning on first launch.

If the app is blocked, open it from Finder with right-click > Open, or allow it from System Settings > Privacy & Security.

## Recovery

If the app exits unexpectedly and closed-lid sleep behavior is not restored, run:

```bash
sudo pmset -c disablesleep 0
```
