# Sleepless 0.1.2

Pre-release security and distribution cleanup for the Sleepless macOS menu bar app.

## Assets

- `Sleepless_0.1.2_aarch64.dmg`
- SHA-256: `c9a2f3764edf441570191c6a54d07f587b0a199d80c51169f3bd9280caa71900`

## Changes

- Rename the internal app package from `ac-awake` to `sleepless`.
- Change the macOS bundle identifier from `app.mac.acawake` to `app.mac.sleepless`.
- Move the privileged helper install path to `/Library/PrivilegedHelperTools/app.mac.sleepless.pmset-helper`.
- Remove the unused Tauri opener plugin and `opener:default` permission.
- Replace the disabled Content Security Policy with a restricted CSP.
- Add `SECURITY.md` with the privileged-helper model, recovery command, and helper removal command.
- Remove internal planning notes from the public repository surface.
- Improve the DMG install window with drag-to-Applications and first-open Gatekeeper guidance.

## Notes

This build is unsigned and unnotarized. macOS may show a security warning on first launch. The DMG install window includes English and Korean first-open guidance, and notes that labels can vary by macOS version.

Older test builds may have installed the previous helper path. To remove it:

```bash
sudo rm -f /Library/PrivilegedHelperTools/app.mac.acawake.pmset-helper
```
